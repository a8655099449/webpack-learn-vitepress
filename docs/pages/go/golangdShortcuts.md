# go lang 快捷键

|      快捷键      |                      作用                      |
| :--------------: | :--------------------------------------------: |
| `Ctrl + + 或 -`  | 可以将当前（光标所在位置）的方法进行展开或折叠 |
|    `Ctrl + j`    |              快速生成一个代码片段              |
|    `alt + j`     |               快速选中相同的内容               |
| `Ctrl + Alt + L` |                   格式化代码                   |
